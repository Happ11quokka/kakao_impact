# 채집권 제거 → 오늘 기록 카운터 전환 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 챗봇·백엔드에서 하루 5회 채집권 시스템을 완전히 제거하고, 원석 저장 완료 시 "오늘 N번째 원석이에요!" 형태로 오늘 기록 횟수를 표시한다.

**Architecture:** chatbot 테이블에서 오늘(KST) 기록 수를 SELECT COUNT로 직접 조회하는 `_db_get_today_count()` 헬퍼를 추가한다. collection_tickets 관련 함수 6개·상수 2개를 삭제하고, 챗봇 핸들러(맞아요/모두 채집/내 원석/채집 안내)와 백엔드 confirm 엔드포인트를 수정한다. DB 테이블(collection_tickets)과 프론트엔드는 이번 작업 범위 밖이다.

**Tech Stack:** Python 3.12, psycopg2, FastAPI, Railway PostgreSQL

      if not RAILWAY_DATABASE_URL:
          return 0
      today = _today_kst()
      try:
          conn = psycopg2.connect(RAILWAY_DATABASE_URL)
          cur = conn.cursor()
          cur.execute(
              """
              SELECT COUNT(*) FROM chatbot
              WHERE user_id = %s
                AND gem != '일상기록'
                AND (created_at AT TIME ZONE 'Asia/Seoul')::date = %s
              """,
              (user_id, today),
          )
          count = cur.fetchone()[0]
          cur.close()
          conn.close()
          return int(count)
      except Exception as e:
          print(f"[_db_get_today_count error] {e}")
          return 0
